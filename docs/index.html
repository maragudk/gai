What it does

GAI provides interfaces to work with:

Chat models

Have conversations with AI models using a simple, consistent interface.

Embedding models

Turn text into vectors for semantic search and similarity operations.

Tools

Extend AI capabilities with custom functions and external integrations.

Evaluations

Test model quality with qualitative and quantitative metrics.

Why use it

You want a unified way to work with AI models. You need a library that:

  • Gives you consistency across providers
  • Keeps implementations simple
  • Makes testing straightforward
  • Uses Go idioms

Getting started

Install the package:

go get maragu.dev/gai

Use it with your favorite models. These client implementations are available:

  • gai-openai
  • gai-google
  • gai-anthropic
